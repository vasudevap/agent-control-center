## Requirements

Every component must:

- Use design tokens only
- Use Auto Layout principles
- Be responsive
- Support light and dark themes
- Meet WCAG 2.2 AA
- Be reusable
- Be documented

---

## States

Where applicable support:

- Default
- Hover
- Focus
- Active
- Disabled
- Loading
- Error
