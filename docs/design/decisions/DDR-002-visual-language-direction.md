# DDR-002 — Visual Language Direction

**Status:** Accepted  
**Date:** 2026-07-10  
**Owner:** Product Design

---

# Decision

Atlas adopts a restrained editorial enterprise visual language.

This decision is based on the Portfolio Admin reference and supplements the previously approved Graticule typography benchmark.

The reference establishes the desired level of visual refinement, spacing, hierarchy, and restraint.

It does **not** define the Atlas color palette, branding, or logo.

---

# Approved Characteristics

## Typography

- Large, confident page titles
- Strong hierarchy
- Inter for primary UI
- JetBrains Mono for technical labels and metadata
- Editorial spacing between headings and content

## Layout

- Generous whitespace
- Calm information density
- Strong alignment
- Consistent card rhythm
- Minimal visual noise

## Cards

- Thin borders
- Minimal shadows
- Larger internal padding
- Premium proportions
- Consistent spacing

## Icons

- Small
- Functional
- Secondary to typography
- Lucide only

## Information Hierarchy

Content should feel curated rather than crowded.

Important information is emphasized through typography and spacing rather than color.

## Color

Do not adopt the warm beige palette from the reference.

Atlas will continue using a cool neutral enterprise palette with a restrained brand accent.

## Overall Feeling

Atlas should feel like:

- an enterprise operating platform
- premium software
- engineered
- calm
- trustworthy
- timeless

It should not feel like:

- a marketing dashboard
- a consumer SaaS application
- an analytics toy
- an AI demo
