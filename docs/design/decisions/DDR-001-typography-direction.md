# DDR-001 — Typography Direction

**Status:** Accepted  
**Date:** 2026-07-10  
**Owner:** Product Design

---

# Decision

Atlas will adopt a typography direction that is clean, modern, highly legible, understated, and optimized for enterprise software.

Typography should communicate professionalism, precision, and operational clarity rather than personality.

---

# Context

During the Visual Identity exploration, the typography used in the **Graticule** concept was selected as the preferred direction.

While the logo mark itself has not been selected, its typographic system establishes the visual tone we want Atlas to convey.

This decision records the desired typography direction and the characteristics that should influence future UI, brand, and documentation work.

---

# Typography System

The Graticule concept uses two complementary typefaces.

## Primary Typeface — Inter

Google Fonts

Recommended weights:

- 400 Regular
- 500 Medium
- 600 SemiBold
- 700 Bold

Inter should become the primary candidate for:

- Product UI
- Dashboard headings
- Navigation
- Buttons
- Tables
- Forms
- Marketing website
- Documentation
- Brand wordmark exploration

The preferred wordmark treatment observed in the concept uses:

- Weight: 600 (SemiBold)
- Letter spacing: approximately -2%

---

## Secondary Typeface — JetBrains Mono

Google Fonts

Recommended weights:

- 400 Regular
- 500 Medium

JetBrains Mono should be evaluated as the secondary system typeface for technical metadata.

Recommended usage:

- Section labels
- Technical annotations
- Component specifications
- Design system references
- Grid labels
- Pixel measurements
- Version information
- Build metadata
- Documentation callouts
- Design review boards

Preferred styling:

- Uppercase
- Wide tracking (approximately 0.2em)
- Reduced opacity
- Small sizes

This treatment creates the restrained engineering "specification sheet" aesthetic that aligns well with Atlas.

---

# Design Objectives

Atlas typography should feel:

- Clean
- Professional
- Modern
- Technical
- Calm
- Premium
- Timeless
- Enterprise-ready

It should support prolonged daily use and remain highly legible across dashboards, dense operational tables, documentation, and technical interfaces.

This decision influences:

- Logo wordmark
- Product UI
- Design System
- Component Library
- Documentation
- Marketing website
- Presentations
- Brand assets

# Reference

Primary visual reference:

**Graticule** concept board reviewed during the Atlas Visual Identity exploration.

The typography system—not the logo mark itself—should be treated as the benchmark.

Specifically:

- Inter establishes the primary product voice.
- JetBrains Mono establishes the technical annotation language.

Together they create a modern enterprise software aesthetic that feels engineered rather than marketed.

# Future Validation

Before final adoption, validate:

- Readability in dense operational tables
- Numeric alignment
- Dashboard hierarchy
- Accessibility
- Dark mode rendering
- Cross-platform consistency
- Browser performance
- Licensing suitability

# Success Criteria

Typography should make Atlas feel like software created by an experienced enterprise product team.

Users should perceive the interface as:

- Trustworthy
- Structured
- Professional
- Modern
- Calm
- Built for serious operational work
